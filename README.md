# tomasantunes-blog
Blog made with Express and React

## How to run
```
Edit config.json, secret-config.json and tomasantunes-blog-frontend/src/config.json
npm install
cd tomasantunes-blog-frontend
npm install
npm run build
cd ..
npm start
Go to localhost:4000
```

## How to make secret-config.json
