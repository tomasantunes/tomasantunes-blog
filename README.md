# tomasantunes-blog
Blog made with Express and React

## How to run
```
Create secret-config.json by copying and renaming secret-config-base.json
Create config.json by copying and renaming config-base.json
Create frontend/src/config.json by copying and renaming frontend/src/config-base.json
npm install
cd frontend
npm install
npm run build
cd ..
npm start
Go to localhost:4000
```
